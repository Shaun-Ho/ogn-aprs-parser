# OGN APRS Parser

A Rust library and Python extension for parsing Open Glider Network (OGN) APRS aircraft beacon messages.

- `core/`: The Rust parsing implementation. See [README](./core/README.md) for more details
- `python/`: Python bindings using PyO3, including generated `.pyi` type stubs. See [README](./pyo3/README.md) for more details
